# GifConverterApp
Created Gif Converter app with GUI (tkinter) and open-cv/pillow for video processing using python programming language. 

# Purpose:
Is to help show glimpse of project demos through Gifs. By converting .mp4 video files into Gifs. Also, demonstrate a short tutorial on how to use the app. Useful project that will be used frequently to demonstrate project details. GIFs are a fun way to express yourself and interpret what you are learning (general way of communication). Overall, this is an app that will be used extensively to demo personal projects.
